# Claude Code

Claude Code is an advanced terminal-based LLM agent designed to act as an infrastructure assistant, offering deep integration with the user's filesystem, shell, and a suite of development tools. It extends the capabilities of large language models into practical, hands-on system management and coding tasks.

Inspired by the "Tech Lead Workflow," Claude Code enables users to define high-level objectives, delegating the execution and intricate details to the agent. This paradigm allows for a more efficient development process, where the LLM handles the operational burden, from code generation and debugging to system configuration and testing.

As an evolution of tooling in the realm of [[Artificial_Intelligence/AI_and_LLMs.md|AI and LLMs]], Claude Code embodies the vision of an [[Concepts/LLM_as_Infrastructure_Assistant.md|LLM as Infrastructure Assistant]], capable of understanding complex instructions and orchestrating diverse tasks across the development environment.
