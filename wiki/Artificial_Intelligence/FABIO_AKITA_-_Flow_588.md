# FABIO AKITA - Flow 588 (Podcast Highlights)

This node provides a technical synthesis of the Flow Podcast episode 588 featuring Fabio Akita, recorded in April 2026.

## Key Technical Takeaways

### 1. The Death of Parameter Wars (2025-2026)
- **Problem**: Scaling from 1B to 1T parameters resulted in diminishing returns (S-Curve).
- **Shift**: Focus moved to "Thinking" (internal reasoning loops) rather than raw model size.
- **Top Models**: Claude Opus 4.6 (noted for discipline), OpenAI GPT-5.1/5.4.

### 2. The Concept of "Harness" (Rédias)
- **Definition**: The tool layer (CLI, harness) is as important as the model itself.
- **Tools**: Claude Code vs OpenAI Codex.
- **Benchmark**: Success is measured by the harness's ability to maintain project state and complete multi-step tasks.

### 3. Local Inference Hardware
- **NVIDIA 5090**: High TFLOPS but limited to 32GB VRAM.
- **AMD Ryzen AI Max**: Lower TFLOPS but 128GB Unified RAM allows running 122B param models (like Qwen 3.5) locally.

### 4. Agentic Workflows
- Akita describes a shift to "Agentic-first" programming, where he acts as a Tech Lead directing AI specialized in specific domains (e.g., handles the HTML/CSS "frictions").
